# AGENTS

This repository is prepared for LLM-agent driven analysis.

## Primary entry points
- Start at [INDEX.md](INDEX.md).
- Then open module indexes:
  - [ibmi-company_system/INDEX.md](ibmi-company_system/INDEX.md)
  - [IBM-i-RPG-Free-CLP-Code/INDEX.md](IBM-i-RPG-Free-CLP-Code/INDEX.md)
  - [intERPrise/INDEX.md](intERPrise/INDEX.md)

## Agent workflow
1. Read root INDEX.md for module map.
2. Read target module INDEX.md for local structure and key files.
3. Read key files first (README, build files, then source files).
4. Produce findings in this order: purpose, dependencies, entry points, risks, next actions.

## Output format recommendation
- Scope analyzed
- Architecture summary
- Important files and why
- Potential issues or unknowns
- Concrete next steps

## Notes
- INDEX.md files are auto-generated snapshots of each folder.
- Re-generate indexes after large structural changes.
